# Titan-Encoder-Based Temporal Point Process for Intermittent Demand Forecasting

## Abstract
Intermittent demand consists of long zero-demand periods interrupted by irregular positive events with highly variable magnitudes. Rather than treating this setting as fixed-interval regression, we formulate it as prediction over a marked temporal event sequence. For each future demand event, the model predicts a demand-size class, an inter-event time, and a continuous residual used to reconstruct the final order quantity. This magnitude-factorized design preserves the interpretability of demand classes while reducing the information loss caused by assigning each class a single representative value.

We first apply this formulation to recurrent marked temporal point process models and then combine it with a Titan sequence encoder, yielding TitanTPP. Experiments on an intermittent spare-parts demand dataset and a weekly aggregated yellow-taxi demand dataset show that the proposed formulation improves quantity reconstruction and that TitanTPP achieves the strongest overall validation performance after convergence. On the taxi-demand dataset, TitanTPP reduces quantity MAE by 13.8% (2827.4 to 2436.1) and lowers validation NLL by 11.5% (1.602 to 1.417) relative to the RMTPP baseline. These results suggest that modeling sparse demand as marked event sequences is effective, and that richer sequence encoders are especially beneficial when demand timing and magnitude are highly heterogeneous.
